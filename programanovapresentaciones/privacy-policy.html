Contact email / Email de contacto: pmongeholgado@gmail.com

Last updated / Última actualización: 10 May 2026

This Privacy Policy explains how the Plataforma Nova app handles user data.

Esta Política de Privacidad explica cómo la aplicación Plataforma Nova trata los datos de usuario.

1. Data collection / Recogida de datos

Plataforma Nova does not require users to create an account and does not intentionally collect personal or sensitive user data through the app.

Plataforma Nova no requiere que los usuarios creen una cuenta y no recoge intencionadamente datos personales o sensibles de los usuarios a través de la aplicación.

2. Data use / Uso de los datos

Since the app does not intentionally collect personal or sensitive user data, no personal data is used for advertising, profiling, resale, or third-party marketing.

Dado que la aplicación no recoge intencionadamente datos personales o sensibles, no se utilizan datos personales para publicidad, perfiles, reventa o marketing de terceros.

3. Data sharing / Compartición de datos

Plataforma Nova does not sell, rent, or share personal user data with third parties.

Plataforma Nova no vende, alquila ni comparte datos personales de los usuarios con terceros.

4. Third-party services / Servicios de terceros

The app may open or display web content related to Plataforma Nova services. External hosting, browser, device, operating system, or platform providers may process technical information according to their own policies.

La aplicación puede abrir o mostrar contenido web relacionado con los servicios de Plataforma Nova. Proveedores externos de alojamiento, navegador, dispositivo, sistema operativo o plataforma pueden procesar información técnica conforme a sus propias políticas.

5. Security / Seguridad

We take reasonable measures to keep the app and related web services secure. The app is intended to use secure HTTPS connections where applicable.

Tomamos medidas razonables para mantener segura la aplicación y los servicios web relacionados. La aplicación está destinada a usar conexiones seguras HTTPS cuando corresponda.

6. Data retention and deletion / Conservación y eliminación de datos

As the app does not intentionally collect personal user data or create user accounts, there is no personal account data retained by the app. If a user contacts the developer by email, the user may request deletion of that communication by writing to: pmongeholgado@gmail.com.

Como la aplicación no recoge intencionadamente datos personales ni crea cuentas de usuario, no existen datos personales de cuenta conservados por la aplicación. Si un usuario contacta con el desarrollador por correo electrónico, puede solicitar la eliminación de esa comunicación escribiendo a: pmongeholgado@gmail.com.

7. Children / Menores

Plataforma Nova is not specifically directed to children. We do not knowingly collect personal data from children.

Plataforma Nova no está dirigida específicamente a menores. No recogemos conscientemente datos personales de menores.

8. Changes to this policy / Cambios en esta política

This Privacy Policy may be updated when necessary. The latest version will always be available on this page.

Esta Política de Privacidad puede actualizarse cuando sea necesario. La última versión estará siempre disponible en esta página.

9. Contact / Contacto

For privacy questions or requests, contact: pmongeholgado@gmail.com.

Para preguntas o solicitudes sobre privacidad, contacta en: pmongeholgado@gmail.com.
