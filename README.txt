 
PROGRAMA NOVA V1.0 — DISTRIBUCIÓN OFICIAL
Autores: Pablo Monge & Nova
Fecha de compilación: 7/nov/2025

1) ¿Qué es?
Herramienta local (sin internet) para generar y presentar contenidos
a partir de un “brief” con estructura, pestañas de navegación y guía hablada.

2) Requisitos
- Windows 10 u 11 (64 bits).
- Python 3.8 o superior instalado (necesario para el servidor local).
  Si no tiene Python, puede descargarlo desde https://www.python.org/downloads/

3) Archivos incluidos
- index.html → interfaz del programa.
- assets\favicon.ico y assets\ProgramaNova_OFICIAL.ico → iconos.
- Iniciar_Programa_Nova.bat → inicia servidor y abre el navegador automáticamente.
- Iniciar_Servidor_Nova.bat → inicia solo el servidor (uso técnico).
- README.txt → este documento.

4) Cómo ejecutarlo (recomendado)
A) Descomprima el ZIP en una carpeta en su escritorio (o donde prefiera).
B) Haga doble clic en: Iniciar_Programa_Nova.bat
   - Se abrirá una ventana negra (servidor).
   - Luego se abrirá automáticamente el navegador en http://localhost:8080
C) Use las pestañas (Brief, Estilo, Estructura, Slides, Exportar).
D) Si cierra el navegador, puede volver a abrirlo entrando a:
   http://localhost:8080/index.html
E) Para detener el servidor: vaya a la ventana negra y pulse Ctrl + C.

5) Notas
- Todo funciona de forma local. No se envían datos a internet.
- La guía hablada usa la síntesis de voz del navegador.
- Si su navegador no abre automáticamente, copie esta dirección en la barra:
  http://localhost:8080/index.html
- Si recibe un aviso del antivirus al abrir los .bat, elija “Permitir”.
  Los .bat solo lanzan Python y abren el navegador.
