DESCRIPCIÓN GENERAL  
Programa Nova es un generador de presentaciones asistido por IA.  
Esta versión (V1.0 Estable) está diseñada para ser usada directamente en la web, sin instalación ni dependencias externas.  
El objetivo es mostrar a los clientes el funcionamiento del sistema de forma clara, segura y fluida.

---

CARACTERÍSTICAS PRINCIPALES  
- Interfaz fija, sin pestañas dinámicas.  
- Sección “Brief del Proyecto” visible permanentemente.  
- Formulario de contacto operativo y página de confirmación (gracias.html).  
- Sincronización automática entre GitHub y Netlify.  
- Código limpio (HTML, CSS, JS) y estructura modular para futuras versiones.
